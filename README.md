# 📝 Funny Todo App

A simple and quirky To-Do list made with React — where even the tasks make you smile! 😄

## 🚀 Features
- Add, delete, and complete tasks
- Fun placeholder tasks like:
  - "Stare at the ceiling and question your existence"
  - "Blame Google for your bugs"
  - "Take a break from your break"

## 🛠️ Tech Stack
- ReactJS
- CSS
- Firebase (optional for backend)
